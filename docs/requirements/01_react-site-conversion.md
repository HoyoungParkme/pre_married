# REQ-01: sample.html → React 정적 웹사이트 전환

- 요청일: 2026-04-15
- 상태: 진행 중
- 스프린트: 1~6

## 요청 원문
> 이 html을 가지고 기능 분리 및 고도화 작업을 통해 진짜 웹사이트 형태로 만들어서 깃허브에 올려서 깃허브 페이지로 작동할거야. 그러니 백엔드 없는 그런 페이지로 변환해줘. 전체적인 디자인도 다시 만들고 React로 만들어줘

## 핵심 목표
- 단일 JSX 파일(sample.html)을 **유지보수 가능한 React 프로젝트**로 분리한다.
- **백엔드 없이** GitHub Pages에서 동작하는 정적 웹사이트로 배포한다.
- 전체 디자인을 리뉴얼하고 기능을 **고도화**한다.

## 사용자 시나리오
1. 예비 신혼부부가 웹사이트 URL에 접속해 초기 자금/전세/저축/결혼 비용을 입력하면 실시간으로 자금 흐름과 최종 잔액을 확인한다.
2. 여러 시나리오(프리셋)를 저장·비교하여 의사결정을 돕는다.
3. 결혼 준비 체크리스트와 혼수 위시리스트로 할 일을 관리한다.
4. 대시보드를 이미지로 저장해 파트너와 공유한다.

| F-8 | 결혼 준비 체크리스트 | 필수 | D-180/90/30/7 단계별 항목 | 미시작 |
| F-9 | 혼수 위시리스트 | 필수 | 항목·가격·상태 CRUD, 총합 반영 | 미시작 |

## 기술 결정
- 빌드: Vite + React + TypeScript
- 스타일: Tailwind CSS (기존 코드 계승)
- 상태: zustand + persist
- 라우팅: react-router-dom HashRouter (GitHub Pages 호환)
- 차트: Recharts
- 내보내기: html-to-image
- 배포: GitHub Actions → GitHub Pages
